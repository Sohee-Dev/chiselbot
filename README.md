## 📘 프로젝트 개요

이 프로젝트는 **AI를 활용한 면접 코칭 서비스**로, 사용자가 면접 질문에 대한 답변을 입력하면

AI가 답변을 분석하고 피드백 및 점수를 제공하는 시스템입니다.

- **사용자용 Flutter 앱** : 면접 코칭, 문의, 계정 관리 등 기능 제공
- **관리자용 SSR 웹 콘솔** : 질문/프롬프트/통계/관리자 계정 관리
- **공통 API 서버 (Spring Boot)** : AI 연동, 데이터 관리, 인증/인가 처리

### 👩‍💻 담당 개발 내용:

- **AI 면접 코칭 시스템**
    - 사용자 답변 → 유사도 분석 → 프롬프트 생성 → AI 피드백 생성 전체 흐름 구현
    - OpenAI `ChatModel` 기반 실시간 코칭
    - 레벨별 전략 패턴으로 다양한 피드백 구조 지원
- **(Admin) 프롬프트 관리**
    - AI코칭에 사용하는 프롬프트 버전관리
    - 레벨별 프롬프트 등록, 수정, 활성화 기능 구현
    - 활성화 시 동일 레벨의 다른 버전 자동 비활성화 처리
- **(Admin)면접 질문 등록**
    - 질문 CRUD + AI학습용 벡터 데이터 관리
    - 난이도(LEVEL)별로 정답 또는 의도/포인트 저장
    - 정답의 경우 임베딩 모델을 사용해 임베딩 후 벡터 저장
- **(Admin) 대시보드 정보 연동**
    - 회원수, 문의 수, 질문 통계 등 주요 지표 시각화
    - Chart.js 기반 그래프 표시 / 실시간 통계데이터 연동
- **보관함 저장 API 구현**
    - 사용자가 받은 AI 패드백 결과를 개인 보관함에 저장/조회/삭제
    - 질문, 답변, 피드백, 유사도 등 학습 기록 관리
- **(Admin) 메뉴관리**
    - 관리자 페이지 메뉴 구조 관리 기능
    - 메뉴 순서 중복 시 자동 정렬 처리
- **(Admin) 관리자 계정 관리**
    - 관리자 계정 등록/조회/삭제/상세보기 기능

## 🤖 AI 피드백 프로세스

1. 사용자가 면접 질문에 답변 입력
2. API 서버에서 해당 질문의 **의도·정답 벡터 데이터 조회**
3. **유사도 분석 (Embedding / Cosine Similarity)**
4. 적합한 **프롬프트 생성 및 AI 요청**
    1. Level1: 모범답안의 유사도와 사용자 답안의 유사도를 비교하여 답안의 부족한부분을 피드백 & 힌트 제공
    2. Level2 ~ Level3: 질문의도와 포인트, 사용자 답안을 Chat모델이 분석 후 피드백 & 힌트 제공
5. **OpenAI / ChatGPT ChatModel**을 통한 실시간 피드백 생성
6. 피드백 및 점수 결과를 사용자에게 실시간 전송

## 🗂️ 질문 유형

| 레벨 | 유형 | 설명 | 전략 |
| --- | --- | --- | --- |
| **Level 1** | 지식형 (Knowledge) | 정의나 개념 중심의 사실 기반 질문 | 모범답안의 유사도와 사용자 답안의 유사도를 비교하여 답안의 부족한부분을 피드백 & 힌트 제공 |
| **Level 2** | 이해형 (Understanding) | 원리나 구조를 설명하는 질문 |  |
| **Level 3** | 응용·사고형 (Application/Thinking) | 실제 상황 적용 및 경험 중심 |  |
